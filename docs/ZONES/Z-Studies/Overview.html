üîÆ Self-Defining Variables (The Magic)

Every zone has access to ALL 103 variables. They define themselves through CSS custom properties:

/* Zone doesn't inherit - it TRANSFORMS */
.zone {
    /* 9 Color Variables */
    color: var(--color);
    background: var(--background);
    border-color: var(--border-color);
    
    /* Size Variables */
    width: var(--width);
    padding: var(--padding);
    
    /* Typography Variables */
    font-size: var(--font-size);
    font-weight: var(--font-weight);
    
    /* ... 90+ more ... */
}

/* Preset SETS the variables */
.hero {
    --font-size: 4rem;    /* Zone becomes large */
    --color: white;       /* Zone becomes white text */
    --background: purple; /* Zone becomes purple */
}

.card {
    --font-size: 1rem;    /* Same zone, now small */
    --padding: 1rem;      /* Same zone, now padded */
    --border-radius: 8px; /* Same zone, now rounded */
}

Key insight: Variables aren't inherited from parents - they're SET by presets. This is why any zone can become anything!
