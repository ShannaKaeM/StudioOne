The Entire System in 5 Parts

1. BASE CLASS
   .zone { /* Maps CSS variables to properties */ }

2. POSITION (Grid Areas)
   .a { --grid-area: a; }
   .b { --grid-area: b; }

3. LAYERS (Z-index)
   .back { --z-index: 0; }
   .front { --z-index: 2; }

4. PRESETS (Transform zones)
   .split { --grid-areas: "a b"; }
   .fullscreen { --grid-areas: "a"; }

5. HTML (Just zones!)
   <div class="zone split">
       <div class="zone a"></div>
       <div class="zone b"></div>
   </div>

How It Works

Split Screen
Grid: "a b"
Zone A = Left
Zone B = Right
Full Screen
Grid: "a"
All zones = Same area
Z-index creates layers
