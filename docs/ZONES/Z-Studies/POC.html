🎯 Zone System - Proof of Concept

The Entire System in 5 Parts

1. BASE CLASS
   .zone { /* Maps CSS variables to properties */ }

2. POSITION (Grid Areas)
   .a { --grid-area: a; }
   .b { --grid-area: b; }

3. LAYERS (Z-index)
   .back { --z-index: 0; }
   .front { --z-index: 2; }

4. PRESETS (Transform zones)
   .split { --grid-areas: "a b"; }
   .fullscreen { --grid-areas: "a"; }

5. HTML (Just zones!)
   <div class="zone split">
       <div class="zone a"></div>
       <div class="zone b"></div>
   </div>

How It Works

Split Screen
Grid: "a b"
Zone A = Left
Zone B = Right
Full Screen
Grid: "a"
All zones = Same area
Z-index creates layers

The Magic: Layering

FULL SCREEN LAYERS (all in grid area "a"):
┌─────────────────────────┐
│ Zone C (front/z:2)      │ ← Content
│ ┌─────────────────────┐ │
│ │ Zone B (middle/z:1) │ │ ← Overlay
│ │ ┌─────────────────┐ │ │
│ │ │ Zone A (back/z:0)│ │ │ ← Background
│ │ └─────────────────┘ │ │
│ └─────────────────────┘ │

All three zones occupy the SAME grid area "a"
but stack using z-index!

Key Insights

  • ✓ Everything is a zone - No semantic HTML needed
  • ✓ Position is math - Grid areas a, b, c, d
  • ✓ Layers via z-index - Stack zones in same area
  • ✓ Presets transform - Same zones, different layouts
  • ✓ ~50 lines of CSS - Entire system!
